# Clustering_shinyapp# Clustering Shiny App

## Overview

The Clustering Shiny App was developed by **Bingtian Ye (Waxman Lab)**. Built upon the Seurat framework, it integrates modules for normalization, scaling, PCA, graph construction, Harmony integration, clustering, and subsetting. The app also provides corresponding visualizations, including UMAP plots, heatmaps, and dot plots.
